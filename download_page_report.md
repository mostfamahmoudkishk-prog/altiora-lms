# Desktop Player Download Page Report
**Component**: `src/routes/app.downloads.player.tsx`

## Overview
A dedicated download portal `/app/downloads/player` has been built to distribute the desktop player installer and instruct students on setup.

## Features
1. **Direct Download Link**:
   - Downloads the `AltioraPlayer-Setup.exe` installer binary.
2. **Luxury Theme Integration**:
   - Styled with the luxury gold and deep grey palette, using glowing shadow cards.
3. **Setup Guidelines**:
   - Displays clear step-by-step guides for installation.
4. **Security Warnings**:
   - Details hardware limit rules, background processes bans, and DRM rules to prepare the student.
